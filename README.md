# Calculator

https://strjakhk.github.io/Calculator/

Last project for The Odin Project Foundations path. It's a simple calculator. Inspired by the Windows Calculator.

## Features

* Basic math operations

* Operations log

## Future changes

* Keyboard input
* Improvement of screen for log and operations (avoid the string overflow in screen)
* Trim left zero if there is no decimals
* on click effects when pressing buttons
* Limit number of decimals allowed in results to avoid screen overflow.
